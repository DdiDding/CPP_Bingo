# 🎯 BINGO

> C/C++와 Windows Console API로 구현한 플레이어 대 컴퓨터 방식의 콘솔 빙고 게임입니다.

일반적인 빙고와 달리, 자신이 선택한 숫자는 내 빙고판에서 **체크**되고 상대방의 같은 숫자는 **봉쇄**됩니다. 단순한 운뿐 아니라 상대의 빙고 완성을 방해하는 선택이 필요한 턴제 대전 게임입니다.

## 주요 기능

- 3×3, 4×4, 5×5 빙고판 지원
- 플레이어와 컴퓨터의 무작위 빙고판 생성
- 플레이어와 컴퓨터가 번갈아 진행하는 턴 시스템
- 선택한 숫자의 중복 및 입력 범위 검사
- 가로·세로·양방향 대각선 빙고 판정
- 기본·체크·봉쇄 상태를 콘솔 색상으로 구분
- 방향키와 Enter 키를 이용한 메뉴 조작
- 승리·패배·무승부 판정
- 게임 재시작 및 메인 메뉴 이동

## 게임 규칙

1. 3×3, 4×4, 5×5 중 원하는 빙고판 크기를 선택합니다.
2. 플레이어와 컴퓨터가 서로 다른 무작위 빙고판을 받습니다.
3. 양쪽이 번갈아 아직 선택되지 않은 숫자를 하나씩 고릅니다.
4. 선택한 숫자는 선택자의 판에서 **체크**되고 상대 판에서는 **봉쇄**됩니다.
5. 체크된 칸으로 가로·세로·대각선 한 줄을 먼저 완성하면 승리합니다.
6. 모든 숫자가 사용될 때까지 빙고가 완성되지 않으면 무승부입니다.

| 표시 상태 | 의미 |
|---|---|
| 흰색 | 아직 선택되지 않은 숫자 |
| 파란색 | 자신의 선택으로 체크된 숫자 |
| 빨간색 | 상대의 선택으로 봉쇄된 숫자 |

## 조작 방법

| 입력 | 동작 |
|---|---|
| `←` `→` | 빙고판 크기 변경 |
| `↑` `↓` | 메뉴 이동 |
| `Enter` | 메뉴 선택 |
| 숫자 입력 | 플레이어 턴에 숫자 선택 |

## 핵심 구현

### 가변 크기 빙고판

판의 한 변 길이를 나타내는 `line_num`을 기준으로 배열 크기와 출력 좌표를 계산합니다. 동일한 게임 로직으로 3×3부터 5×5까지의 판을 처리합니다.

### 게임 데이터와 출력 데이터 분리

플레이어와 컴퓨터의 판은 각각 판정용 배열과 출력용 배열로 관리합니다. 판정용 배열의 값을 상태 값으로 변경하고, 출력용 배열에는 원래 숫자를 보존하여 선택된 숫자와 상태를 함께 보여 줍니다.

```cpp
0      // 체크된 칸
-100   // 봉쇄된 칸
양수   // 아직 선택되지 않은 칸
```

### 빙고 판정

1차원 배열을 2차원 보드처럼 사용하며 다음 인덱스 규칙으로 한 줄의 완성 여부를 검사합니다.

- 가로: 연속된 인덱스 순회
- 세로: `line_num` 간격으로 순회
- 정방향 대각선: `line_num + 1` 간격으로 순회
- 역방향 대각선: `line_num - 1` 간격으로 순회

### 선택 이력 관리

플레이어와 컴퓨터가 선택한 숫자를 하나의 배열에 저장합니다. 새 숫자를 선택할 때 이 배열을 조회하여 이미 사용한 숫자가 다시 선택되는 것을 방지합니다.

### 콘솔 인터페이스

Windows Console API를 사용해 커서 위치와 글자 색상을 제어합니다.

- `SetConsoleCursorPosition`: UI 요소를 지정한 좌표에 배치
- `SetConsoleTextAttribute`: 빙고판 상태별 색상 적용
- `_getch`: 방향키와 Enter 입력 처리

```text
       ↓

```
